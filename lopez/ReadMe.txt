Lopez CD++, 
Originally designed by Alejandro López. 
Improved by Sixuan Wang.

Lopez-src folder is the src code
Lopez-test folderis the test models

Lopez is an extended version of CD++ for Cell-DEVS, allowing the cells to use multiple state variables and to use  multiple  ports  for  inter-cell  communications. Current, Lopez works fine in GCC4.xx or higher in Linux,

More information can be found here:
A  López,  G.  Wainer.  2003.  Extending  CD++  Specification  Language  for  Cell-DEVS  Model 
Definition.
