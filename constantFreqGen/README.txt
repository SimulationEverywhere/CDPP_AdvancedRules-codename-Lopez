Este modelo está creado para ser compilado con el simulador CD++ de la versión Simu Lopez
Autor: Laouen Mayal Louan belloli
Fecha: 02/09/2016

Nombre: constantFreqGen
Descripción

El modelo envia el output 1 a una frecuancia constante de 1 segundo por el puerto out.
Cuando le entra cualquier mensaje por el puerto stop, el modelo se pasiva dejando de enviar outputs.
Cuando le entra cualquier mensaje por el puerto start, el modelo retoma la tarea de enviar el output 1 
a la misma frecuencia de 1 segundo.
